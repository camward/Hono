# Hono

## Server

```bash
# перейти в папку server
$ cd server

# установить зависимости
$ npm i

# поднять postgres
$ docker-compose up

# запустить приложение
$ npm run dev
```

Сервер будет доступен по адресу `http://localhost:3000/`

## Client

```bash
# перейти в папку client
$ cd client

# установить зависимости
$ npm i

# запустить приложение
$ npm run dev
```

Приложение будет доступно по адресу `http://localhost:8080/`
